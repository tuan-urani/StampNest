Để chạy được ứng dụng thì cần cài:
1, npm install
2, npm run dev

Cần thay đổi API trong file src/lib/api.ts

3, để buil được Zalo Mini App thì các bạn cần cài zmp cli -> cấp quyền login -> tạo zalo mini app để lấy id và thay đổi trong app-config.json

4, mỗi lần build mới thì cần chạy lệnh zmp build hoặc npm run build để build www -> thay đổi config -> zmp deploy.
